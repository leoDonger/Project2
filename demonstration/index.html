Write up

The score is displayed on the bottom left. A pinball earns 50 points when it bounces on a non-center circular obstacle and 100 points for the center circular obstacle.

Game setting: Collision sound effects when scoring. Pinballs bounce off each other. Default starting with 3 pinballs. A particle system display appears when a pinball is near a flipper.

Press "space" to start or pause the game and "r" to reset the game.

Difficulties I encountered: not being able to create a rotated rectangle with processing with my Box class; collision equations in general; and obstacle texturing is hard in processing
